# Staging View (Transaction List)

The staging view displays imported transactions for review and editing before final submission.

## Components

1. **Transaction Table** - Main table showing all staged transactions
2. **Category Assignment** - Category and subcategory pickers
3. **Transaction Editing** - Editable descriptions, account, bank selection
4. **Wanted Level** - "I Wanted It" classification
5. **Submit Button** - Validates and submits transactions

## Purpose

This view enables users to:
- Review imported transactions
- Assign categories and subcategories
- Edit transaction descriptions
- Set "I Wanted It" levels
- Select account and bank for each transaction
- Delete unwanted transactions
- Submit validated transactions to permanent storage
