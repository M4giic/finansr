# Import Screen

The Import Screen is the main interface for importing financial transactions into the application. It consists of two primary components that work together to provide a complete import workflow.

## Components

1. **Import Wizard** - Multi-step wizard for uploading and configuring transaction imports
2. **Staging View** (Transaction List) - Review and edit staged transactions before final submission

## Purpose

This screen enables users to:
- Import transactions from CSV files (mBank, Citi Handlowy)
- Configure import parameters (account, bank, date range)
- Detect and handle duplicate transactions
- Track import coverage to prevent gaps or overlaps
- Review and edit transactions before final submission
- Submit transactions to permanent storage and Google Sheets
