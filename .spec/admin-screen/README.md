# Admin Screen

The Admin screen provides management interfaces for accounts and categories.

## Components

1. **Account Manager** - Create, view, and delete accounts
2. **Category Manager** - View, edit, and delete categories with subcategories

## Purpose

This screen enables users to:
- Manage multiple accounts (Personal, Joint)
- View all categories grouped by account
- Edit category names
- Manage subcategories hierarchically
- Delete unused categories/subcategories
